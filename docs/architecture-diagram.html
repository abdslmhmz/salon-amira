Salon Amira — System Architecture

Booking system · FastAPI + React + MySQL 8.4 · Docker Compose

Frontend Stack

  • React 18 + Vite (dev server)
  • Nginx reverse proxy (prod)
  • Source Sans 3 typography
  • Glassmorphism nav (backdrop-blur)
  • 4 pages: Landing, Booking, Dashboard, Lookup
  • Custom Calendar.jsx (zero deps)

Backend Core

  • FastAPI + Uvicorn (port 8000)
  • Slot calculation: date overlap + overrides
  • Appointment CRUD with status machine
  • PDF ticket generation (ReportLab A5)
  • Phone lookup for existing appointments
  • SQLAlchemy ORM with MySQL

Data Layer

  • MySQL 8.4 (port 3307 external)
  • 5 tables: services, availabilities,
  • appointments, schedule_overrides,
  • blocked_slots
  • Healthcheck: mysqladmin ping
  • Init SQL via docker-entrypoint
