## Variant: Calm Editorial

### Design stance
Sparse, elegant, typography-first. Like reading a fashion editorial — large
headlines, generous whitespace, weight-300 Source Sans 3, Stripe blue shadows.
Every element earns its place. No decoration without purpose.

### Key choices
- Layout: Single column, centered at 700px (hero) / 1000px (services)
- Typography: Source Sans 3 weight 300 throughout. h1 at 3rem, tight letter-spacing
- Color: Purple (#533afd) as sole accent. Rose (#c44569) absent from this variant
- Shadows: Stripe-style blue-tinted multi-layer
- Cards: White with soft shadow, no borders
- Nav: Glass morphism (backdrop-blur)
- Hero: Centered, badge + headline + subtext + 2 CTAs + stats row

### Trade-offs
- Strong at: Reading flow, premium feel, trust signals
- Weak at: Information density, quick scanning for power users

### Best for
First-time visitors discovering the brand. Marketing-focused. Best matches the
current Salon Amira design direction.
