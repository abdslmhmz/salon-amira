## Variant: Playful Gradient

### Design stance
Warm, feminine, beauty-brand. Rose-to-purple gradients, pill-shaped buttons,
rounded cards, soft glowing shadows. Like Glossier or Sephora's digital presence.
Emotional, inviting, makes you want to book.

### Key choices
- Layout: Centered hero with floating stats. Pill cards for services
- Typography: Source Sans 3, mixed weights — 300 for hero headline, 600-700 for CTAs
- Color: Rose (#c44569) as primary accent. Purple (#7c3aed) as secondary.
  Rose-to-purple gradients throughout
- Cards: Rounded (20px border-radius), pill-shaped price tags, circle icons
- Shadows: Pink-tinted (rgba(196,69,105,...)) — warm, not blue
- Hero: Gradient background with radial glow orbs (pseudo-elements)
- CTA Section: Full-width rose-to-purple gradient with white CTA
- Why Us: Gradient section background + gradient icon circles

### Trade-offs
- Strong at: Brand warmth, emotional connection, feminine aesthetic
- Weak at: Information density, professional/technical feel, accessibility
  (gradients behind text need care)

### Best for
First-time visitors on mobile. Instagram/TikTok traffic. Beauty-conscious
audience that values aesthetics and emotional appeal.
