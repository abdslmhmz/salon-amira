## Variant: Utilitarian Dense

### Design stance
Tool-like, information-dense, action-first. Like a well-designed SaaS dashboard.
Compact cards, live availability data in the hero, status bar, fast scanning.

### Key choices
- Layout: Two-column hero with live slots sidebar. Dense 7-card grid
- Typography: Source Sans 3 weight 400-700. Smaller sizes (0.8-0.95rem body)
- Color: Accent border on nav (#533afd 2px). Green availability indicators
- Cards: Thin bordered cards (no shadows). Compact padding (16px)
- Nav: Flat, no glassmorphism. Bold logo + segmented tabs
- Hero: Split layout — text left, live availability slots right
- Status bar showing open hours + location

### Trade-offs
- Strong at: Quick booking, power users, information scanning
- Weak at: Emotional connection, brand warmth, premium feel

### Best for
Returning clients who know what they want. Provider-side dashboard users.
